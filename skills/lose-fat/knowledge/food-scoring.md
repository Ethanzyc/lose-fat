# 食物评分系统

## 评分原理

每种食物每标准份量对应一个固定分数。分数主要反映热量，同时叠加营养权重调整。

**计算公式：**

- 基础分 = 标准份量热量(kcal) / 100，向上取整
- 营养调整：高蛋白(>15g/份) -0.5、高纤维(>3g/份) -0.5、高饱和脂肪(>5g/份) +1、含添加糖(>10g/份) +1
- 最终得分最低 0.5 分

**灯色分类：**

- 🟢 绿灯：≤ 3 分且营养密度高 → 优先选择，在份量内自由搭配
- 🟡 黄灯：3-5 分或绿灯食物超量 → 适量食用，控制频率
- 🔴 红灯：> 5 分或高饱和脂肪/高添加糖 → 尽量避免

## 使用指南

1. **先看灯色**：优先选择 🟢 绿灯食物，🟡 黄灯适量，🔴 红灯尽量避免
2. **再看分数**：在灯色允许的前提下，通过组合让每餐总分达到目标范围
3. **不要求精确**：每餐分数在目标范围上下浮动 1 分完全没问题
4. **注意份量**：分数基于标准份量，如果吃的量翻倍，分数也要翻倍
5. **烹饪方式**：同一种食材，清蒸/水煮比油炸/红烧分数更低（额外加油会增加分数）
6. **外食怎么办**：大致估算，选择清淡菜品，控制总量。一餐外食如果超过 10 分，下一餐适当减少
7. **未列出的食物**：参考同类食物的分数，或用「热量/100 向上取整」作为基础分估算

## 放纵餐（Cheat Meal）

### 为什么需要放纵餐

坚持性是减脂成功的第一要素。每周安排 1-2 顿计划内的放纵餐，比完全禁止所有"不健康"食物更容易长期坚持。完全禁止某类食物反而容易触发暴食。

### 放纵餐基本原则

- **频率**：每周 1-2 顿，建议安排在运动日或周末
- **时间限制**：放纵餐 ≠ 放纵日，控制在一顿（1-1.5 小时内吃完）
- **心态**：这是计划的一部分，不是"犯错"，享受完立刻回到正轨
- **策略**：先吃蔬菜和蛋白质打底，再享受想吃的食物，自然控制总量

### 放纵餐后的体重反弹

放纵餐后第二天称重可能会发现体重增加 1-2 kg，**这几乎全是水分，不是脂肪**：

1. 外食通常高盐高钠，一顿可能摄入 3000-5000mg 钠
2. 身体必须维持固定的血液钠浓度（渗透压平衡），多余的钠需要额外锁水来稀释
3. 每多摄入 1g 钠，身体潴留约 1L 水
4. 碳水摄入增加也会带来糖原和结合水分（每 1g 糖原绑定约 3g 水）
5. 恢复清淡饮食后 2-3 天，多余水分自然排出，体重回落

**关键心态**：不要因为一顿放纵餐后的体重波动而沮丧，这是正常的生理反应，继续执行计划即可。

### 一顿放纵餐到底会长多少脂肪

| 放纵餐类型 | 估算额外热量 | 可能的脂肪增加 |
|-----------|------------|--------------|
| 正常放纵（多吃了 500-800 kcal） | +500-800 kcal | 约 70-110g（0.07-0.11 kg） |
| 大吃一顿（多吃了 1000-1500 kcal） | +1000-1500 kcal | 约 140-210g（0.14-0.21 kg） |
| 自助餐吃到撑（多吃了 2000+ kcal） | +2000+ kcal | 约 280g+（0.28 kg+） |

> **换算逻辑：** 1 kg 脂肪 ≈ 7700 kcal。多吃了 1000 kcal，理论上最多增加约 130g 脂肪。第二天体重增加的 1-2 kg 中，脂肪只占很小一部分。

### 火锅怎么吃

火锅是减脂期最适合的放纵餐之一——因为你可以完全控制吃什么。关键在于选择正确的锅底、蘸料和食材。

#### 锅底选择

| 锅底 | 推荐度 | 原因 |
|------|--------|------|
| 清汤/菌汤/番茄锅 | 🟢 首选 | 热量最低，不额外加油 |
| 骨汤/椰子鸡 | 🟢 | 相对清淡 |
| 牛油辣锅 | 🟡 可以吃 | 脂肪含量高（表面厚厚一层油），但可以捞掉表面的油 |
| 鸳鸯锅 | 🟢 | 辣锅涮味道，清汤锅涮食材（减少吸油） |
| 咖喱锅/冬阴功 | 🟡 | 含椰浆或较多油 |
| 麻辣烫/串串香锅底 | 🔴 避免 | 反复煮沸的油，脂肪和嘌呤极高 |

> **辣锅减油技巧：** 涮好的食材先在清汤锅或白开水里过一下再吃，可以减少 30-50% 的表面油脂。

#### 蘸料策略

蘸料是火锅隐形热量的最大来源：

| 蘸料 | 热量/勺（约 15g） | 建议 |
|------|------------------|------|
| 芝麻酱 | ~90 kcal | 🔴 一碗芝麻酱 ≈ 3-4 碗米饭的热量 |
| 沙茶酱 | ~75 kcal | 🔴 |
| 蒜泥香油碟 | ~120 kcal | 🔴 一勺纯油 |
| 香油 + 蚝油 | ~80 kcal | 🔴 |
| 海鲜酱油 + 醋 | ~10 kcal | 🟢 首选 |
| 海鲜酱油 + 醋 + 蒜泥 + 香菜 | ~12 kcal | 🟢 最推荐 |
| 小米辣 + 酱油 + 醋 | ~10 kcal | 🟢 想吃辣选这个 |

**蘸料建议：** 用酱油+醋+蒜泥+香菜+小米辣调一碗底料，最多加半勺芝麻酱提味。

#### 食材选择

**🟢 放心吃（高蛋白低脂，火锅必备）：**

| 类别 | 食材 | 说明 |
|------|------|------|
| 肉类 | 牛肉片（瘦）、羊肉片（瘦） | 火锅最好的蛋白质来源 |
| 肉类 | 毛肚、百叶、鸭肠 | 高蛋白低脂，涮 10-15 秒即可 |
| 海鲜 | 虾滑、鱼片、鱿鱼 | |
| 豆制品 | 老豆腐、腐竹（少量） | 腐竹脂肪较高，控制量 |
| 蔬菜 | 生菜、菠菜、白菜、茼蒿 | 多吃，几乎零热量 |
| 蔬菜 | 冬瓜、白萝卜、金针菇 | |
| 菌类 | 香菇、杏鲍菇、平菇 | |
| 主食 | 魔芋丝、粉丝（少量） | 魔芋丝几乎零热量 |

**🟡 控制量：**

| 类别 | 食材 | 控制原因 |
|------|------|---------|
| 加工肉类 | 午餐肉、火腿肠、培根 | 高钠 + 高饱和脂肪 |
| 丸类 | 各种丸子（鱼丸、肉丸、虾丸） | 淀粉含量高，真正肉含量低 |
| 主食 | 方便面、乌冬面、年糕 | 碳水炸弹，吸汤后热量翻倍 |
| 豆制品 | 油豆腐、炸腐竹 | 已经炸过，脂肪极高 |
| 内脏 | 猪脑、鹅肠 | 胆固醇极高 |

**🔴 尽量避免：**

| 类别 | 食材 | 原因 |
|------|------|------|
| 高脂肉类 | 五花肉、肥牛卷（雪花多的） | 脂肪含量 30-50% |
| 油炸类 | 酥肉、小酥肉、炸馒头 | 吸满油脂 |
| 甜品类 | 红糖糍粑、炸汤圆 | 高糖高脂 |
| 饮品 | 啤酒、可乐、酸梅汤（加糖） | 液体热量 |

#### 火锅进餐顺序

1. **先喝一碗清汤** — 垫底增加饱腹感
2. **先涮蔬菜和菌类** — 用蔬菜填饱一半的胃
3. **再涮肉类和海鲜** — 充分享受蛋白质
4. **最后吃主食** — 到这一步通常已经七八分饱，主食自然吃得少

#### 火锅后的补救

| 措施 | 原因 |
|------|------|
| 多喝白开水 | 促进钠排出 |
| 次日清淡饮食 | 低盐、多蔬菜、优质蛋白 |
| 不用刻意节食 | 饿了反而容易连续放纵 |
| 2-3 天后称重 | 刚吃完称到的都是水重 |

### 烧烤怎么吃

烧烤的主要问题：大量油脂、高温产生的有害物质（杂环胺、多环芳烃）、高钠调味。但烧烤同样可以吃得相对健康。

#### 烧烤策略

| 策略 | 做法 | 原因 |
|------|------|------|
| 先点一份凉菜 | 拍黄瓜、凉拌木耳、毛豆 | 打底增加饱腹感 |
| 主攻肉类 | 多点瘦肉串，少点肥肉串 | 蛋白质为主 |
| 多点蔬菜 | 烤韭菜、烤茄子、烤金针菇、烤青椒 | 增加纤维 |
| 要求少刷油 | 烧烤师傅刷油非常大方，主动要求少油 | 减少大量隐形油脂 |
| 去掉焦糊部分 | 烤焦的部分不要吃 | 减少致癌物摄入 |
| 不要点主食类 | 烤馒头、烤面包片、炒粉 = 额外碳水 + 额外油 | 专注于蛋白质和蔬菜 |

| 🟢 | 毛豆、花生（少量） | |
| 🟡 | 烤五花肉、烤培根卷 | 脂肪含量高，每次限 2-3 串解馋 |
| 🟡 | 烤香肠、烤热狗 | 加工肉类，高钠 |
| 🔴 | 烤馒头片、烤面包片、炒粉 | 纯碳水 + 大量油 |
| 🔴 | 烤脑花 | 胆固醇极高 |
| 🔴 | 啤酒（大量） | 烧烤配啤酒是增肥经典组合 |

#### 烧烤 + 饮品的陷阱

烧烤配啤酒的组合危害叠加：

| 组合 | 额外热量 | 说明 |
|------|---------|------|
| 3 瓶啤酒（330ml × 3） | ~420 kcal | 相当于多吃了一顿饭 |
| 2 瓶啤酒 + 烤馒头片 2 个 | ~600 kcal | |
| 烧烤全程 + 4 瓶啤酒 | ~800+ kcal | 仅饮品就超过半天热量预算 |

**减脂期烧烤饮品建议：** 白开水、苏打水、无糖茶。

### 自助餐怎么吃

自助餐是减脂期最大的挑战，但掌握策略后也可以应对：

#### 自助餐三层策略

**第一层：侦察（前 5 分钟）**

- 绕场一圈，看清楚所有菜品种类
- 心里挑好想吃的 3-5 样（不要超过 5 样）
- 不要急于拿盘子装

**第二层：策略取餐**

| 步骤 | 做法 | 原因 |
|------|------|------|
| 第 1 盘 | 沙拉/凉拌菜（装满一盘） | 先用蔬菜填饱胃 |
| 第 2 盘 | 选 2-3 样最想吃的菜（每样少量） | 每样先尝一小份，好吃再拿 |
| 第 3 盘 | 选 1-2 样想再吃的 | 到这里通常已经饱了 |

**第三层：防线**

| 原则 | 做法 |
|------|------|
| 每样只拿一小份 | 一口能尝到味道的量就够了 |
| 不要回本心态 | 自助餐吃回本的代价是多余的热量 |
| 坐在离取餐区远的位置 | 减少视觉诱惑和起身次数 |
| 吃完等 10 分钟再决定是否拿更多 | 饱腹信号需要 15-20 分钟到达大脑 |

#### 自助餐避雷

| 区域 | 建议 |
|------|------|
| 生鲜/海鲜区 | 🟢 多拿，高蛋白低脂 |
| 刺身/寿司区 | 🟢 刺身放心吃；寿司注意饭量 |
| 烤肉区 | 🟢 瘦肉为主 |
| 沙拉区 | 🟢 首选，但注意酱汁（选油醋汁） |
| 热菜区 | 🟡 选 1-2 样尝味道 |
| 主食区 | 🟡 少量，或跳过 |
| 甜品区 | 🔴 选 1 小份最想吃的，细品慢吃 |
| 饮品区 | 🔴 避开果汁和汽水 |

### 其他常见放纵场景

#### 朋友聚餐

| 场景 | 策略 |
|------|------|
| 川菜/湘菜 | 多点一道清炒时蔬；辣菜用水涮一下再吃；主食减半 |
| 日料 | 刺身、烤物为主；少点天妇罗等油炸；寿司控制在 6-8 个 |
| 西餐 | 牛排选菲力/西冷（瘦）；沙拉代替薯条配菜；酱汁分开 |
| 烤肉店 | 同烧烤策略；生菜包肉代替直接吃（增加纤维） |
| 家常菜馆 | 点 1 荤 2 素 1 汤；米饭控制半碗到一碗 |

#### 节假日

| 节日 | 挑战 | 应对 |
|------|------|------|
| 春节 | 连续聚餐 + 零食 | 每顿吃到七分饱；零食选坚果和水果；每天保持运动 |
| 中秋 | 月饼 | 一个月饼切成 4-8 份，尝味道即可（一个广式月饼约 800 kcal） |
| 端午 | 粽子 | 一个粽子当主食，减少其他碳水；选肉粽比豆沙粽好 |
| 元宵/汤圆 | 汤圆 | 3-4 个汤圆当主食（一碗约 300-400 kcal） |
| 生日/聚会 | 蛋糕 | 一小角（切 1/10）细细品尝，不重复拿 |

#### 下午茶/奶茶

| 选择 | 建议 |
|------|------|
| 奶茶 | 无糖 + 鲜奶茶底（非植脂末）；加料选仙草/寒天（不要珍珠/芋圆） |
| 咖啡 | 美式/拿铁（无糖）；避免焦糖玛奇朵等特调 |
| 蛋糕 | 选一小块，当天不再吃其他甜品 |
| 水果茶 | 糖含量极高（即使是"纯水果"），不如直接吃水果 |

### 放纵餐总结原则

| 原则 | 说明 |
|------|------|
| 计划内，不内疚 | 提前安排好哪一顿是放纵餐，吃完不后悔 |
| 一顿，不是一天 | 放纵餐限制在 1-1.5 小时，下一顿立刻恢复 |
| 蛋白质打底 | 先吃蔬菜和蛋白质，再享受想吃的 |
| 选一样最想吃 | 与其每样都吃一点，不如选一样最爱好好享受 |
| 多喝水 | 帮助代谢和排出多余钠 |
| 次日正常吃 | 不要因为"昨天吃多了今天就别吃了"而节食，这反而引发暴食循环 |
| 关注周趋势 | 一顿放纵餐不会毁掉一周的成果，看每周平均体重趋势 |
