# 饮食营养指南

## 热量计算

### TDEE 计算公式（Mifflin-St Jeor）

**男性：** BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 + 5
**女性：** BMR = 10 × 体重(kg) + 6.25 × 身高(cm) - 5 × 年龄 - 161

**TDEE = BMR × 活动系数**

| 活动水平 | 系数 |
|---------|------|
| 久坐（几乎不运动） | 1.2 |
| 轻度活动（每周 1-3 次） | 1.375 |
| 中度活动（每周 3-5 次） | 1.55 |
| 高度活动（每周 6-7 次） | 1.725 |

### 减脂热量缺口

- 建议每日热量缺口：300-500 kcal
- 每周减重目标：0.5-1 kg
- 男性每日摄入不低于 1500 kcal，女性不低于 1200 kcal

## 三大营养素分配

| 营养素 | 占比 | 说明 |
|--------|------|------|
| 碳水化合物 | 40-45% | 以全谷物、杂粮为主，严格限制精制糖 |
| 蛋白质 | 25-30% | 优先选择优质蛋白，保护肝功能 |
| 脂肪 | 25-30% | 以不饱和脂肪为主，限制饱和脂肪 < 7% |

## 食物红黑榜

### 红榜（推荐）

| 类别 | 食物 |
|------|------|
| 主食 | 糙米、燕麦、全麦面包、红薯、玉米、藜麦 |
| 蛋白质 | 鸡胸肉、鱼虾、豆腐、鸡蛋、去皮鸡腿肉 |
| 蔬菜 | 西兰花、菠菜、芹菜、黄瓜、番茄、生菜、苦瓜 |
| 水果（低糖） | 苹果、柚子、蓝莓、草莓、猕猴桃 |
| 脂肪来源 | 橄榄油、牛油果、坚果（每日一小把） |
| 其他 | 绿茶、黑咖啡（无糖） |

### 黑榜（避免或严格限制）

| 类别 | 食物 | 原因 |
|------|------|------|
| 主食 | 白米饭（大量）、白面包、油条、糕点 | 高 GI、快速升糖 |
| 蛋白质 | 肥肉、五花肉、油炸肉类、加工肉制品 | 高饱和脂肪 |
| 饮品 | 含糖饮料、果汁、啤酒、白酒 | 果糖和酒精直接促进脂肪合成 |
| 零食 | 薯片、饼干、冰淇淋、糖果 | 高糖高脂 |
| 调味 | 大量沙拉酱、蚝油（含糖高） | 隐形热量 |

## 水果选择指南

> 水果是健康饮食的一部分，但对 TG 偏高和脂肪肝人群，**不是所有水果都适合随便吃**。选对水果、吃对方式很重要。

### 水果分三类

#### 🟢 推荐每天吃（低糖高纤维）

每天 1-2 份（200-350g），作为加餐或餐间零食。

| 水果 | 每 100g 糖分 | 每 100g 热量 | 特点 |
|------|------------|------------|------|
| 草莓 | ~5g | ~32 kcal | 维生素 C 丰富 |
| 蓝莓 | ~10g | ~57 kcal | 抗氧化物质丰富 |
| 柚子 | ~7g | ~42 kcal | 饱腹感强、水分足 |
| 猕猴桃 | ~9g | ~56 kcal | 维生素 C + 纤维 |
| 苹果 | ~10g | ~52 kcal | 果胶丰富，饱腹感好 |
| 橙子 | ~9g | ~48 kcal | |
| 番茄（大） | ~3g | ~18 kcal | 可当水果可当菜 |
| 黄瓜 | ~2g | ~15 kcal | 随便吃 |
| 番石榴 | ~9g | ~41 kcal | 纤维含量极高 |
| 柠檬 | ~2g | ~29 kcal | 泡水、调味 |

#### 🟡 控制份量（糖分较高或容易吃多）

每次限 1 份（约 150-200g），每天不超过 1 次。

| 水果 | 每 100g 糖分 | 每 100g 热量 | 控制原因 |
|------|------------|------------|---------|
| 香蕉 | ~12g | ~89 kcal | 糖较高，运动前吃更好 |
| 葡萄 | ~16g | ~69 kcal | 糖高 + 无饱腹感 + 一吃停不下来 |
| 西瓜 | ~6g | ~30 kcal | 糖虽不高但 GI 高（76），吃太快太多 |
| 哈密瓜 | ~8g | ~34 kcal | GI 较高 |
| 芒果 | ~14g | ~60 kcal | 果糖含量高，TG 高者少吃 |
| 荔枝 | ~15g | ~66 kcal | 高糖 + 容易上火 + 一次吃很多 |
| 桂圆（鲜） | ~16g | ~70 kcal | 同荔枝 |
| 樱桃 | ~13g | ~63 kcal | 糖不低，容易不知不觉吃半斤 |
| 柿子 | ~14g | ~74 kcal | |
| 石榴 | ~14g | ~83 kcal | |

#### 🔴 减脂期不建议吃（高糖高热量）

| 水果 | 每 100g 糖分 | 每 100g 热量 | 不建议原因 |
|------|------------|------------|-----------|
| 椰肉（鲜） | ~6g | ~354 kcal | 脂肪含量高达 33g/100g |
| 牛油果 | ~1g | ~160 kcal | 健康脂肪但热量极高，每天最多半个 |
| 榴莲 | ~27g | ~147 kcal | 高糖高脂，一小块 200g 就 300 kcal |
| 菠萝蜜 | ~19g | ~94 kcal | 糖极高，黏腻口感让人停不下来 |
| 冬枣 | ~20g | ~105 kcal | 糖极高，"枣"不等于健康 |
| 红枣（干） | ~60g | ~281 kcal | 浓缩糖炸弹，当零食吃极容易超标 |
| 葡萄干 | ~60g | ~299 kcal | 同上 |
| 柿饼 | ~48g | ~250 kcal | 脱水后糖分浓缩 |
| 果脯/蜜饯 | ~50-65g | ~300+ kcal | 加糖腌制，比糖果还甜 |
| 水果罐头 | ~15-20g（糖水） | ~50-80 kcal | 泡在糖水里 |

> **关键原则：** 水果干的糖分是鲜果的 3-6 倍（脱水浓缩），且丧失了饱腹感。一把葡萄干（30g）≈ 一大串鲜葡萄的糖分，但完全吃不饱。

### 为什么绝对不能榨汁

> **果汁 = 去掉了所有好东西的糖水。** 无论鲜榨还是瓶装，对 TG 偏高人群都是有害的。

#### 榨汁过程中发生了什么

| 步骤 | 结果 |
|------|------|
| 纤维被过滤/打碎 | 失去了延缓糖分吸收的"刹车" |
| 细胞壁被破坏 | 糖分从"被包裹"变为"游离"状态 |
| 维生素接触空气氧化 | 维生素 C 等损失 30-50% |
| 多个水果浓缩为一杯 | 3 个橙子 ≈ 1 杯橙汁，但 3 个橙子你吃不下，1 杯果汁一口闷 |
| 果糖直接冲击肝脏 | 无纤维缓冲，肝脏瞬间收到大量果糖 |

#### 橙子 vs 橙汁对比

| 指标 | 吃 1 个橙子 | 喝 1 杯橙汁（300ml，约 3 个橙子） |
|------|-----------|-------------------------------|
| 果糖 | ~6g | ~18g |
| 膳食纤维 | ~3g | ~0.5g（滤掉大部分） |
| 吃/喝的时间 | 3-5 分钟，需要咀嚼 | 30 秒喝完 |
| 饱腹感 | 较强（纤维 + 咀嚼） | 几乎没有 |
| 血糖上升速度 | 缓慢（GI ~43） | 快速（GI ~65） |
| 肝脏脂肪合成刺激 | 低 | 高（大量果糖同时涌入） |

#### 果汁为什么特别伤害 TG 和脂肪肝

1. **果糖直达肝脏：** 果汁中的果糖没有纤维延缓，快速被吸收入血，全部由肝脏代谢
2. **促进脂肪新生（DNL）：** 肝脏处理大量果糖时，会将多余部分转化为脂肪 → 直接加重脂肪肝
3. **没有饱腹感 → 喝完还想吃：** 喝一杯 300 kcal 的果汁不影响你接下来吃饭，但吃 3 个橙子你肯定吃不下饭了
4. **果糖不触发饱腹信号：** 葡萄糖会刺激胰岛素和瘦素分泌让你感觉"饱了"，果糖不会 → 你喝再多也不觉得饱

#### "鲜榨"也不行

| 常见误区 | 事实 |
|---------|------|
| "自己在家鲜榨的就很健康" | 去掉纤维的问题一样存在，只是没有额外添加糖 |
| "不滤渣的果汁保留了纤维" | 打碎后的纤维结构被破坏，延缓吸收的效果大打折扣 |
| "加蔬菜一起榨就好了" | 确实比纯果汁好，但打碎的纤维效果仍不如直接吃 |
| "果汁排毒/排毒养颜" | 没有任何科学依据，"排毒"是营销概念 |
| "破壁机保留了全部营养" | 破壁只是打碎，不能恢复纤维的物理结构 |

#### 正确做法

| 方式 | 推荐度 | 说明 |
|------|--------|------|
| 直接吃整个水果 | 🟢 首选 | 完整保留纤维，天然控制摄入速度 |
| 水果切块拌酸奶 | 🟢 | 增加蛋白质和钙 |
| 水果 + 坚果 | 🟢 | 健康脂肪 + 纤维，饱腹感强 |
| 果蔬打碎不过滤 | 🟡 可接受 | 比纯果汁好，但不如直接吃 |
| 纯果汁 | 🔴 避免 | |
| 市售果汁饮料 | 🔴 避免 | 可能额外加糖 + 防腐剂 |

## 饮食方案

> **注意：** 本项目使用食物评分系统，不再使用固定食谱模板。
> 详细的评分方法和食物列表请参考 `food-scoring.md`。

每餐的基本结构建议（具体搭配按评分系统执行）：

- **早餐**：全谷物主食 + 蛋白质 + 蔬果
- **午餐**：杂粮主食 + 瘦肉/鱼 + 蔬菜
- **晚餐**：减少主食或用薯类替代 + 蛋白质 + 大量蔬菜
- **加餐**（可选）：坚果或低糖水果

## 饮食注意事项

1. 每日饮水 2000-2500ml
2. 进食顺序：蔬菜 → 蛋白质 → 主食
3. 每餐咀嚼充分，用餐时间 ≥ 20 分钟
4. 晚餐在 19:00 前完成，睡前 3 小时不进食
5. 外食时优先选择清蒸、水煮、凉拌的菜品

## 渗透压与水潴留

### 为什么一顿重口味就能「吃回」一周的成果？

很多人在严格控制饮食一周后，吃了一顿重口味饭菜，第二天发现体重反弹 1-2 kg，感到沮丧。但这个反弹绝大多数不是脂肪。

### 钠与水潴留的原理

1. 每摄入 1g 钠（约 2.5g 食盐），身体会额外潴留约 1000ml 水分
2. 一顿重盐饮食的钠含量可达 3000-5000mg（相当于 7.5-12.5g 盐），远超每日建议量（< 2000mg 钠 = 5g 盐）
3. 结果：一顿高钠饮食可能导致身体额外潴留 1-2 升水 = 体重增加 1-2 kg

### 如何区分水重和脂重

| 特征 | 水重（水分变化） | 脂重（脂肪变化） |
|------|----------------|----------------|
| 变化速度 | 快（1-3 天内波动） | 慢（需要数周） |
| 波动幅度 | 1-3 kg | 每周 0.3-0.8 kg |
| 触发因素 | 高钠饮食、生理期、碳水摄入增加 | 持续热量盈余/缺口 |
| 恢复速度 | 2-4 天恢复正常 | 需要持续努力 |
| 伴随症状 | 可能感觉浮肿、手指发紧 | 无明显感觉 |

### 日常建议

1. **不要被单次称重吓到**：关注每周平均体重趋势，而非某一天的数值
2. **高钠饮食后多喝水**：充足的水分反而有助于排出多余的钠
3. **注意隐形钠**：酱油（1 勺约含钠 1000mg）、豆瓣酱、腌制食品、加工食品
4. **称重条件统一**：每周固定时间（建议周一晨起、排便后、空腹、穿相同衣物）
5. **心态调整**：体重在一天内波动 1-2 kg 是完全正常的，不代表你「吃胖了」

### 什么情况需要关注

- 体重持续 2 周以上呈上升趋势（不是短期波动）→ 需要重新审视饮食评分执行情况
- 伴随明显水肿（按压小腿凹陷不回弹）→ 建议就医检查

### 参考数据

> - 世界卫生组织建议每日钠摄入量 < 2000mg（约 5g 盐）
> - 中国居民平均每日食盐摄入约 10.5g，远超建议量
> - 减脂期间建议每日食盐控制在 5g 以内（约一啤酒瓶盖的量）
