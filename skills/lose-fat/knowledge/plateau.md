# 平台期识别与突破

## 什么是平台期

体重连续 **2 周以上** 没有明显变化（波动 < 0.3 kg），且饮食和运动执行到位时，称为平台期。

> 1 周体重不变不代表平台期。正常的体重波动范围为 ±0.5-1 kg。

## 平台期成因

### 1. 代谢适应（最主要原因）

减重过程中，身体会通过多种机制降低能量消耗来对抗体重下降：

| 适应机制 | 影响幅度 |
|---------|---------|
| 基础代谢下降（BMR 降低） | 每减 10 kg，BMR 约下降 100-150 kcal/天 |
| 非运动活动热效应（NEAT）降低 | 减重期 NEAT 可自发减少 200-400 kcal/天 |
| 食物热效应降低 | 进食量减少 → 消化耗能减少 50-100 kcal/天 |
| 运动效率提高 | 同样运动消耗的热量随适应而减少 10-20% |

**综合影响：** 减重 10 kg 后，每日总消耗可能比初始减少 400-700 kcal，远超单纯 BMR 计算的预期。

### 2. 水分波动掩盖脂肪减少

| 因素 | 影响 |
|------|------|
| 高钠饮食 | 潴留 1-2 kg 水分 |
| 运动后肌肉修复 | 炎症反应可暂时增加 0.5-1 kg |
| 生理周期（女性） | 经前可增加 1-3 kg |
| 压力/睡眠不足 | 皮质醇升高导致水潴留 |

> 这时候脂肪确实在减少，但被水分增加抵消了。继续坚持，通常 1-2 周后会出现"掉秤"。

### 3. 饮食偏差积累

- 分量估算逐渐放松（不知不觉多吃了 10-20%）
- 调味料/酱汁的热量被低估
- 零食/饮料的隐性热量

### 4. 运动适应

- 做同样的运动时间、同样的强度，身体消耗的热量比初期少 10-20%
- 需要调整运动方式、强度或时长

## 诊断：是真平台还是假平台

| 检查项 | 真平台 | 假平台（水分掩盖） |
|--------|--------|------------------|
| 7 天移动平均趋势 | 持平或上升 | 持平但每日波动大 |
| 腰围测量 | 没变化 | 持续减小 |
| 运动表现 | 没有提升 | 在提升（说明身体在变） |
| 饮食执行 | 严格 | 近期有高钠/放纵餐 |
| 衣服合身度 | 没变化 | 变松了 |

**如果腰围在减小、运动表现有提升、衣服变松了，说明脂肪在减少，不需要任何干预，继续坚持即可。**

## 突破策略

### 策略 1：饮食休息（Diet Break）

**原理：** 暂时恢复到维持热量 1-2 周，让代谢激素（瘦素、甲状腺激素）部分恢复，心理上也获得喘息。

| 参数 | 建议 |
|------|------|
| 时长 | 7-14 天 |
| 热量 | 恢复到 TDEE（维持体重热量） |
| 营养素 | 蛋白质保持不变，增加碳水 |
| 预期体重 | 可能增加 0.5-1.5 kg（水分和糖原，不是脂肪） |
| 恢复减脂后 | 通常 1-2 周内体重会快速下降到低于平台期水平 |

**适用人群：** 已持续减脂 8 周以上、心理疲劳感明显、严格执行饮食但体重不动的人。

### 策略 2：碳水补充日（Refeed Day）

**原理：** 单日大幅增加碳水摄入，刺激瘦素分泌和代谢率短期提升。

| 参数 | 建议 |
|------|------|
| 频率 | 每 1-2 周 1 次 |
| 热量 | 比减脂热量高 30-50%（主要增加碳水） |
| 碳水目标 | 当天碳水 3-5 g/kg 体重 |
| 蛋白/脂肪 | 保持正常水平 |
| 食物选择 | 米饭、红薯、燕麦、水果等优质碳水（不是蛋糕奶茶） |
| 次日预期 | 体重增加 0.5-1 kg（糖原+水分），2-3 天恢复 |

**适用人群：** 低碳水饮食者、减脂 4-6 周后进入平台的人。

### 策略 3：提升 NEAT

NEAT（非运动活动热效应）是平台期最容易被忽视的变量。减脂过程中 NEAT 会不自觉下降，每日可差 200-400 kcal。

| 行为 | 额外消耗 |
|------|---------|
| 每天 10000 步（vs 5000 步） | +150-250 kcal |
| 走楼梯代替电梯（5 层/天） | +30-50 kcal |
| 站立办公 2 小时（vs 坐着） | +50-80 kcal |
| 做家务 30 分钟 | +80-120 kcal |
| 饭后散步 15 分钟 | +40-60 kcal |

> 这些看似微小的活动累加起来，每天可以额外消耗 300-500 kcal，相当于一次中等强度运动。

### 策略 4：调整运动方案

| 调整方向 | 具体做法 |
|---------|---------|
| 增加强度 | 快走 → 走跑交替；增加力量训练负重 |
| 增加时长 | 有氧从 30 分钟增加到 40-45 分钟 |
| 改变方式 | 引入新的运动形式（游泳、骑车、跳绳） |
| 增加频率 | 从 3-4 天/周增加到 4-5 天/周 |

### 策略 5：重新计算热量需求

体重下降后 TDEE 会降低。如果一直沿用初始的热量目标，缺口可能已经很小甚至消失。

| 体重变化 | 建议操作 |
|---------|---------|
| 减轻 5 kg 以内 | 可能不需要调整 |
| 减轻 5-10 kg | 重新计算 BMR/TDEE，确认热量缺口 |
| 减轻 10 kg 以上 | 必须重新计算，可能需要减少 100-200 kcal/天 |

## 执行优先级

遇到平台期时，按以下顺序排查和调整：

1. **确认是否真平台** — 检查腰围、运动表现、衣服合身度
2. **回顾饮食记录** — 排除隐性热量偏差
3. **提升 NEAT** — 最容易实施、最安全
4. **调整运动** — 增加强度或改变方式
5. **重新计算热量** — 确认缺口是否还够
6. **尝试 Refeed Day** — 每 1-2 周 1 次碳水补充
7. **考虑 Diet Break** — 持续 8+ 周以上且心理疲劳时
