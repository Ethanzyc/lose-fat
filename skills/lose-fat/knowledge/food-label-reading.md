# 食品标签阅读指南

> 学会看营养成分表和配料表，是减脂过程中最实用的技能之一。超市里 90% 的"健康食品"都经不起标签检验。

## 营养成分表

### 强制标示的 "4+1"

中国 GB 28050《预包装食品营养标签通则》要求所有预包装食品必须标示以下 5 项：

| 项目 | 含义 | 单位 |
|------|------|------|
| 能量 | 食品的总热量 | kJ（千焦） |
| 蛋白质 | 蛋白质含量 | g |
| 脂肪 | 脂肪含量 | g |
| 碳水化合物 | 碳水化合物含量 | g |
| 钠 | 钠含量 | mg |

> **kJ 换算 kcal：** 1 kcal ≈ 4.184 kJ。快速心算：kJ ÷ 4.2 ≈ kcal。例如 840 kJ ≈ 200 kcal。

### NRV% — 每日参考值百分比

NRV% 告诉你：吃 100g（或 1 份）这个食品，摄入的营养素占一天推荐量的百分之几。

**常用 NRV 参考值：**

| 营养素 | 每日参考值（NRV） |
|--------|-------------------|
| 能量 | 8400 kJ（约 2000 kcal） |
| 蛋白质 | 60 g |
| 脂肪 | 60 g |
| 碳水化合物 | 300 g |
| 钠 | 2000 mg |
| 膳食纤维 | 25 g |

**实战示例：** 某饼干每 100g 含钠 800mg → NRV% = 800 ÷ 2000 = **40%**。意味着吃 100g 这种饼干，就吃掉了一天近一半的钠配额。

### 快速判断法

拿到一包食品，看营养成分表时关注这 3 个数字：

| 关注项 | 判断标准 | 说明 |
|--------|---------|------|
| 能量 | 每 100g > 400 kcal → 🟡 | 相当于每口都在吃高热量 |
| 脂肪 | 每 100g > 20g → 🟡 | 高脂食品，控制份量 |
| 钠 | 每 100g NRV% > 30% → 🟡 | 高钠食品，注意水潴留 |

> 减脂人群额外关注：**碳水化合物总量**（不只是"糖"）。很多"0糖"产品碳水含量并不低。

### 注意份量陷阱

营养成分表通常标注"每 100g"或"每份"。

**常见套路：**

| 套路 | 示例 |
|------|------|
| 缩小"一份"的定义 | 一包薯片 120g，标注"每份 30g"的热量，看起来不高 |
| 用小包装暗示低热量 | 一瓶饮料 500ml，标注"每 100ml"的热量 |
| 用 kJ 代替 kcal | 1000 kJ 看着比 240 kcal"小"，其实一样 |

**心算法则：** 看清是"每 100g"还是"每份"→ 如果是每份，算出整包的总热量。

## 配料表

### 核心规则：按含量降序排列

配料表中的原料按加入量**从多到少**排列。排在前面的就是含量最多的。

**快速判断：**

| 配料表前三位 | 判断 |
|-------------|------|
| 水、某某粉、某某油 | 加工食品，营养价值有限 |
| 白砂糖 / 果葡糖浆 / 麦芽糊精 | 糖是主要成分，减脂期不推荐 |
| 全麦粉 / 燕麦 / 牛奶 | 相对健康的基底 |

### 隐形糖的 50 个名字

食品工业用不同名称隐藏糖。以下成分**本质上都是糖**：

| 类别 | 常见名称 |
|------|---------|
| 蔗糖类 | 白砂糖、冰糖、红糖、黑糖、绵白糖 |
| 果糖类 | 果葡糖浆、高果糖玉米糖浆（HFCS）、果糖、结晶果糖 |
| 葡萄糖类 | 葡萄糖浆、麦芽糖浆、玉米糖浆、麦芽糊精 |
| 糖醇类（有甜味但有热量） | 麦芽糖醇、山梨糖醇、木糖醇 |
| "天然"糖 | 浓缩果汁、蜂蜜、枫糖浆、龙舌兰糖浆、椰子糖、枣泥 |
| 其他 | 糊精、转化糖浆、焦糖、淀粉糖 |

> **关键区别：** 糖醇类（木糖醇、麦芽糖醇等）热量比蔗糖低，但仍有一定热量和升糖作用；麦芽糖醇的 GI 约 35，热量约蔗糖的 75%。

### 反式脂肪的隐蔽身份

配料表中出现以下成分时，可能含有反式脂肪：

| 配料名称 | 常见于 |
|---------|--------|
| 氢化植物油 | 奶茶奶精、咖啡伴侣 |
| 人造奶油 / 人造黄油 | 蛋糕、饼干、面包 |
| 起酥油 | 派、酥皮点心、油炸食品 |
| 植脂末 | 速溶咖啡、奶茶粉、热可可 |
| 代可可脂 | 巧克力、巧克力涂层食品 |
| 精炼植物油 | 部分糕点、零食 |

> **"0 反式脂肪" ≠ 没有：** 国标规定每 100g 反式脂肪 ≤ 0.3g 即可标注为"0"。少量存在但合法标"0"。

### 添加剂的判断

配料表中较长的化学名称通常是食品添加剂。**合法使用的食品添加剂安全性有保障**，减脂期真正需要关注的不是添加剂，而是：

- 添加糖的含量
- 钠含量（很多调味零食钠极高）
| 总热量密度

## "0 糖 0 脂 0 添加" 营销陷阱

### 陷阱 1："0 糖"但高碳水

国标定义：糖含量 ≤ 0.5g/100g 即可标注"无糖"。

**问题在于"糖"在国标中仅指单糖和双糖**，不包括多糖（淀粉、糊精等）。

| 产品 | 标注 | 实际情况 |
|------|------|---------|
| 某无糖饼干 | "无蔗糖" | 用麦芽糊精替代，GI 高达 85-105（比白糖还高） |
| 某无糖麦片 | "0 添加蔗糖" | 含大量果葡糖浆和糊精 |
| 某无糖酸奶 | "0 蔗糖" | 含果葡糖浆或浓缩果汁 |

> **核心原则：** 看到"0 糖"不要只看标注，翻到营养成分表看**碳水化合物总量**。碳水高但糖低 → 大概率加了麦芽糊精等隐形糖。

### 陷阱 2："0 脂"但高糖

脱去脂肪的食品通常口感变差，厂商会用大量糖来弥补。

| 产品 | 脂肪 | 糖/碳水 | 实际热量 |
|------|------|---------|---------|
| 某脱脂酸奶 | 0.1g | 15g 糖/100g | 约 80 kcal |
| 普通酸奶 | 3g | 8g 糖/100g | 约 70 kcal |
| 某低脂饼干 | 0.5g | 65g 碳水/100g | 约 450 kcal |

> 脱脂 ≠ 低热量。很多"0 脂"食品的热量反而比普通版更高。

### 陷阱 3：代糖的利与弊

| 代糖 | 甜度（相对蔗糖） | 热量 | 安全性评估 |
|------|----------------|------|-----------|
| 赤藓糖醇 | 70% | ~0.2 kcal/g | WHO 认为安全；2023 年《Nature Medicine》研究发现高水平赤藓糖醇与心血管事件风险相关，尚有争议 |
| 三氯蔗糖 | 600 倍 | 0 | 可能影响肠道菌群 |
| 阿斯巴甜 | 200 倍 | 4 kcal/g（用量极少） | IARC 列为"可能致癌物"（2B 类），但 JECFA 认为正常摄入量安全 |
| 甜菊糖苷 | 200-300 倍 | 0 | 植物提取，目前认为较安全 |
| 木糖醇 | 100% | 2.4 kcal/g | 过量可能腹泻（> 50g/天） |
| 麦芽糖醇 | 90% | 2.1 kcal/g | **不是零热量**，GI 约 35，大量食用仍影响血糖 |

**WHO 2023 年建议：** 不应使用非糖甜味剂来控制体重。长期使用代糖可能：
- 无法帮助减重（可能通过味觉补偿机制增加食欲）
- 改变肠道菌群
- 增加 2 型糖尿病风险

**务实建议：**
- 短期替代含糖饮料（从全糖可乐 → 无糖可乐）是可以接受的过渡
- 但最终目标应该是减少对甜味的依赖（从无糖可乐 → 水/茶）
- 不建议长期大量食用代糖食品

### 陷阱 4："0 添加" ≠ 健康

"0 添加蔗糖"、"0 添加防腐剂"、"0 添加色素"……每个声称只针对特定成分：

| 声称 | 只意味着 | 不意味着 |
|------|---------|---------|
| 0 添加蔗糖 | 没额外加白砂糖 | 可能加了果葡糖浆、蜂蜜、浓缩果汁 |
| 0 添加防腐剂 | 没加防腐剂 | 可能含大量糖、钠、反式脂肪 |
| 0 添加色素 | 没加人工色素 | 可能加了焦糖色（本身也是添加剂） |
| 纯天然 | — | 没有法律定义，无监管含义 |

## GI（血糖生成指数）

### 什么是 GI

GI 衡量的是吃下含 50g 碳水的某种食物后，血糖升高的速度和程度。以葡萄糖为 100 作为参照。

| GI 分级 | 数值范围 | 血糖反应 |
|---------|---------|---------|
| 低 GI | ≤ 55 | 血糖缓慢升高，缓慢回落 |
| 中 GI | 56-69 | 血糖中度升高 |
| 高 GI | ≥ 70 | 血糖快速飙升，随后快速回落 |

### 常见食物 GI 值

| 食物 | GI | 分类 |
|------|-----|------|
| 葡萄糖 | 100 | 高 |
| 白米饭 | 83 | 高 |
| 白面包 | 75 | 高 |
| 麦芽糊精 | 85-105 | 高 |
| 白糖（蔗糖） | 65 | 中 |
| 全麦面包 | 50-60 | 中 |
| 燕麦片（传统） | 55 | 中 |
| 红薯 | 44-61 | 低~中 |
| 苹果 | 36 | 低 |
| 牛奶 | 31 | 低 |
| 豆类（扁豆、鹰嘴豆） | 28-40 | 低 |
| 西瓜 | 76 | 高（经常被误判为低 GI） |

### 低 GI 的陷阱

**陷阱 1：低 GI ≠ 低热量**

| 食物 | GI | 每 100g 热量 |
|------|-----|-------------|
| 巧克力（黑巧 70%） | 23（低） | ~600 kcal |
| 油炸食品 | 30-40（低） | 400-600 kcal |
| 白米饭 | 83（高） | 130 kcal |

> 脂肪会延缓胃排空，降低 GI。所以高脂食品往往 GI 低，但热量极高。**减脂期看热量第一，GI 第二。**

**陷阱 2：低 GI 食品不等于适合减脂**

市面上的"低 GI"认证食品：
- 中国有团体标准 T/CNFIA 085-2022 进行低 GI 食品认证
- 但认证仅关注 GI 值，不限制热量、脂肪和添加糖
- 低 GI 饼干、低 GI 蛋糕依然是高热量加工食品

**陷阱 3：GI 受烹饪和搭配影响**

同一食物，不同烹饪方式和搭配的 GI 差异很大：

| 因素 | 影响 | 示例 |
|------|------|------|
| 烹饪程度 | 越煮越烂 GI 越高 | 热土豆 GI 78 → 放凉后 GI 56 |
| 搭配蛋白质/脂肪 | 降低 GI | 白米饭 GI 83 → 米饭+鸡肉+蔬菜 GI ~60 |
| 搭配纤维 | 降低 GI | 白面包 GI 75 → 全麦面包 GI 50 |
| 搭配醋/柠檬汁 | 降低 GI | 醋拌凉面的 GI 比普通面低 20-30% |

### GI 对减脂的意义

对 TG 偏高和脂肪肝人群，低 GI 饮食确实有好处：

- 降低餐后血糖波动 → 减少胰岛素分泌 → 减少脂肪合成
- 更持久的饱腹感 → 减少饥饿和零食摄入
- 改善胰岛素敏感性 → 打破 TG-脂肪肝恶性循环

**但 GI 只是工具之一，不是唯一标准。** 选择食物时：热量和营养密度 > GI 值。

## 实战：30 秒判断一个食品能不能吃

拿到一包食品，按以下顺序快速判断：

### 第一步：看配料表前三位（5 秒）

- 前 3 位有糖/糖浆/糊精 → 🟡 控制量或不选
- 前 3 位是天然食材 → 🟢 继续看

### 第二步：看营养成分表（10 秒）

- 看"每份"还是"每 100g"→ 换算成你要吃的总量
- 能量 ÷ 4.2 = kcal → 评估这包/这份的总热量
- 碳水 > 60g/100g → 🟡 高碳水，控制份量
- 钠 NRV% > 30% → 🟡 高钠，可能潴留水分

### 第三步：判断（15 秒）

| 热量/份 | 判断 |
|---------|------|
| < 150 kcal | 可以作为加餐或一餐的搭配 |
| 150-300 kcal | 可以作为一餐的一部分 |
| 300-500 kcal | 相当于一顿正餐的热量 |
| > 500 kcal | 慎重，这可能是你半天热量预算 |

### 示例

**某品牌"全麦"面包：**
- 配料表：小麦粉、水、白砂糖、起酥油…… → 第一位不是全麦粉，且含糖和反式脂肪来源
- 营养成分：每 100g 350 kcal，碳水 55g，脂肪 8g
- 判断：🟡 这是"假全麦"，减脂期不建议作为主食

**某品牌原味燕麦片：**
- 配料表：燕麦片 → 只有一种原料
- 营养成分：每 100g 370 kcal，碳水 60g（但纤维 10g），蛋白质 13g
- 判断：🟢 真正的燕麦片，碳水虽高但纤维丰富，升糖慢

**某品牌无糖苏打饼干：**
- 配料表：小麦粉、麦芽糊精、植物油、食用盐…… → 第二位就是麦芽糊精
- 营养成分：每 100g 450 kcal，碳水 70g，糖 0.3g
- 判断：🔴 虽然"无糖"，但碳水极高、热量极高、含隐形糖（麦芽糊精）

## 常见"健康食品"打假清单

| 食品 | 营销卖点 | 真实情况 |
|------|---------|---------|
| 果蔬脆片 | "非油炸""纯天然" | 大多数是真空油炸，脂肪含量 15-30%，热量 450-550 kcal/100g |
| 风味酸奶 | "含益生菌""助消化" | 每瓶含糖 10-15g，相当于 3-4 块方糖 |
| 坚果棒/能量棒 | "高蛋白""代餐" | 热量 200-300 kcal/根，糖浆粘合，碳水 30g+ |
| 100% 果汁 | "无添加""纯天然" | 去掉了纤维，浓缩了果糖，GI 接近含糖饮料 |
| 粗粮饼干 | "高纤维""粗粮" | 纤维 3-6g/100g 但脂肪 15-25g，为了口感加了大量油 |
| 0 脂沙拉酱 | "0 脂肪""健康" | 用糖和淀粉替代油脂，热量并不低 |
| 椰子水 | "天然电解质" | 糖含量 3-6g/100ml，一瓶 300ml 相当于 10-18g 糖 |
