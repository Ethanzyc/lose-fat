# 外食与外卖减脂策略

## 现实背景

中国城市 25-50 岁上班族平均每周外食/外卖 5-10 顿。外食的主要问题：

| 问题 | 具体表现 |
|------|---------|
| 油量过大 | 餐馆炒菜用油量是自炊的 3-5 倍（一盘炒青菜可能含 30-40g 油） |
| 钠超标 | 外卖平均单餐含盐 5-8g，已超过全天推荐量（< 5g） |
| 碳水比例高 | 盖浇饭、面条等主食占比常达 60-70%（推荐 40-45%） |
| 隐形糖 | 红烧、糖醋、咖喱类菜肴含大量添加糖 |
| 蔬菜不足 | 外卖蔬菜量通常远低于推荐（每餐 ≥ 200g） |

## 外食原则

### 选店三原则

1. **优先有蒸/煮/炖菜的店** — 粤菜、日料、轻食店通常比川菜、烧烤更容易控制
2. **能选套餐就不选单点** — 套餐结构更完整（主食 + 蛋白 + 蔬菜），不容易偏食
3. **备注"少油少盐"** — 很多商家会配合，实测可减少 20-30% 的用油量

### 点餐四步法

1. **先点蔬菜** — 凉拌/白灼蔬菜 1-2 份（保证 ≥ 200g）
2. **再点蛋白质** — 清蒸/白灼/水煮类的鱼、虾、鸡、豆腐
3. **最后点主食** — 杂粮饭 > 白米饭 > 炒饭/炒面，控制 1 碗以内
4. **少点一个菜** — 外食习惯性多点，实际减少 1 个菜通常刚好够吃

### 烹饪方式选择

| 推荐程度 | 烹饪方式 | 说明 |
|---------|---------|------|
| 优先选择 | 清蒸、白灼、水煮、凉拌、炖汤 | 几乎不额外加油 |
| 可以选择 | 清炒、爆炒、烤 | 油量适中，注意控量 |
| 尽量避免 | 红烧、干锅、油炸、糖醋、干煸 | 油糖含量极高 |

## 外食控量技巧

### 减油

- 菜盘底部有油汤时，米饭不要浇汁
- 能用水涮的就涮一下（可减少 10-15% 油脂）
- 炒菜类去浮油：将菜品在碗边刮一下
- 麻辣烫/冒菜：选清汤底，少喝汤

### 减盐

- 少点腌制类（腊肉、咸鱼、泡菜）
- 汤少喝（外卖汤普遍高钠）
- 酱料分装，自己控制用量
- 不点"下饭"类菜（本质是高盐高油刺激食欲）

### 减碳水

- 米饭只盛半碗或选择小份
- 不点炒饭、炒面等"碳水+油"组合
- 如果已有粉面主食，就不再加米饭/馒头
- 把部分主食换成根茎类蔬菜（芋头、山药）

## 特殊场景应对

### 聚餐/应酬

| 策略 | 说明 |
|------|------|
| 提前吃点东西 | 赴宴前 30 分钟吃个苹果或喝杯无糖酸奶，降低饥饿感 |
| 先吃蔬菜和蛋白质 | 从凉菜和清淡菜开始，减少后续高热量菜品的摄入 |
| 饮料控制 | 以茶水/苏打水替代酒精和含糖饮料 |
| 控制节奏 | 放慢进食速度，每口咀嚼 15-20 次 |
| 不清盘 | 感觉 7-8 分饱就停，不因为"浪费"而过食 |

### 加班夜宵

- 便利店：茶叶蛋 + 关东煮（白萝卜、魔芋、海带）+ 无糖茶 ≈ 150 kcal
- 水果店：一份水果（苹果/橙子）≈ 80 kcal
- 快餐店：粥 + 小菜 ≈ 200 kcal
- 避免：烧烤、炸鸡、方便面、奶茶（单顿 500-800+ kcal）

### 出差旅行

- 高铁/飞机：自带全麦面包 + 坚果 + 水果
- 酒店：选含粥/蒸蛋的早餐，避开油条/煎饼/炒面
- 异地就餐：优先当地清淡特色菜（如粤菜的白切鸡、浙菜的清蒸鱼）
