# 补充剂循证分析

> 本文件提供常见补充剂的循证信息，仅供参考。补充剂不能替代药物治疗和生活方式干预。

## 评级标准

| 评级 | 含义 |
|------|------|
| A | 有高质量 RCT/系统综述支持，获益明确 |
| B | 有多项临床研究支持，获益较可能 |
| C | 有部分研究支持，证据尚不充分 |
| D | 证据不足或结果矛盾 |

## 针对脂肪肝/高 TG 的常见补充剂

### Omega-3 脂肪酸（鱼油）

| 项目 | 内容 |
|------|------|
| 证据等级 | B |
| 有效成分 | EPA + DHA |
| 推荐剂量 | 每天 2-4 g EPA+DHA（降 TG 剂量） |
| 效果 | 可降低 TG 15-30%，对脂肪肝有一定改善 |
| 起效时间 | 8-12 周 |
| 注意事项 | 高剂量可能增加出血风险；与抗凝药合用需谨慎 |
| 食物来源 | 深海鱼（三文鱼、鲭鱼、沙丁鱼）每周 2-3 次可替代补充剂 |

> **优先从食物获取。** 每周吃 2-3 次深海鱼（每次 150g）即可获得约 2-3 g EPA+DHA。只有在无法保证鱼类摄入时才考虑补充剂。

### 奶蓟草（水飞蓟素/Silymarin）

| 项目 | 内容 |
|------|------|
| 证据等级 | B |
| 有效成分 | 水飞蓟素（Silymarin） |
| 推荐剂量 | 水飞蓟素 140-420 mg/天 |
| 效果 | 抗氧化、保护肝细胞膜，部分研究显示可降低 ALT 10-20% |
| 起效时间 | 4-12 周 |
| 注意事项 | 效果因人而异，质量参差不齐；不能替代生活方式干预 |

> **定位：辅助保肝，不是治疗手段。** 对已经控制饮食和运动的人来说，边际收益有限。

### 小檗碱（Berberine）

| 项目 | 内容 |
|------|------|
| 证据等级 | B |
| 来源 | 黄连、黄柏等中药提取 |
| 推荐剂量 | 500 mg × 2-3 次/天（餐中或餐后） |
| 效果 | 可降低 TG 15-25%、空腹血糖 10-20%、改善胰岛素敏感性 |
| 起效时间 | 4-8 周 |
| 注意事项 | 可能引起胃肠道不适（腹泻、便秘）；可能影响其他药物代谢（CYP3A4 酶抑制） |

> **小檗碱是补充剂中证据最强的降 TG/降糖成分之一。** 但因为它会干扰药物代谢，正在服用他汀类、降压药或其他处方药的用户不建议自行使用。

### 维生素 E

| 项目 | 内容 |
|------|------|
| 证据等级 | B（仅限非糖尿病 NAFLD 患者） |
| 推荐剂量 | 800 IU/天（需在医生指导下） |
| 效果 | PIVENS 试验显示可改善非酒精性脂肪肝（NASH）的肝脏组织学 |
| 注意事项 | 高剂量长期使用（> 400 IU/天）可能与全因死亡率增加、出血性卒中风险增加相关 |
| 适用人群 | 仅限非糖尿病、经肝活检确认的 NASH 患者 |

> **不建议自行服用高剂量维生素 E。** 食物来源（坚果、种子、植物油）更安全。

### 姜黄素（Curcumin）

| 项目 | 内容 |
|------|------|
| 证据等级 | C |
| 推荐剂量 | 500-1000 mg/天（需配黑胡椒提取物提高吸收） |
| 效果 | 抗炎、抗氧化，部分研究显示可降低 ALT 和肝脏脂肪含量 |
| 注意事项 | 吸收率低，需要特殊配方（如磷脂复合物） |

### 维生素 D

| 项目 | 内容 |
|------|------|
| 证据等级 | C |
| 推荐剂量 | 缺乏者补充 1000-2000 IU/天 |
| 效果 | 脂肪肝患者维生素 D 缺乏率高达 50-70%。补充可能改善胰岛素敏感性 |
| 检测建议 | 建议先检测 25(OH)D 水平，缺乏（< 20 ng/mL）时再补充 |

### 膳食纤维补充剂

| 类型 | 推荐度 | 说明 |
|------|--------|------|
| 洋车前子壳（Psyllium） | B | 可溶性纤维，可降低餐后血糖和胆固醇，建议 5-10 g/天，需足量饮水 |
| 菊粉（Inulin） | C | 可溶性纤维，促进肠道益生菌，但过量可能引起腹胀 |
| 抗性糊精 | C | 水溶性膳食纤维，耐受性较好 |

> **优先从食物获取纤维。** 补充剂只是在无法通过饮食达标时的辅助手段。

## 不推荐的补充剂

| 补充剂 | 原因 |
|--------|------|
| 各类"排毒"产品 | 无科学依据，可能含泻药成分 |
| 左旋肉碱（L-Carnitine） | 对正常饮食者补充无额外减脂效果 |
| 共轭亚油酸（CLA） | 研究结果不一致，效果微弱 |
| 白芸豆提取物 | 仅阻断部分淀粉吸收，效果有限，不能抵消过量碳水摄入 |
| 酵素/酶类 | 口服后被消化分解为氨基酸，无法发挥酶的功能 |

## 总结建议

1. **补充剂是锦上添花，不是核心。** 饮食控制 + 运动的效果远超任何补充剂
2. **如果只选一种：** Omega-3（尤其不吃鱼的用户），证据最充分、风险最低
3. **已确诊脂肪肝且 TG 持续偏高：** 可考虑小檗碱（需告知医生）
4. **所有补充剂都应告知主治医生**，避免与处方药产生相互作用
5. **购买渠道：** 选择正规品牌，注意查看有效成分含量和第三方检测认证
