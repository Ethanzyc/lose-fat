# 中式健康烹饪指南

## 烹饪方式热量对比

同一种食材，不同烹饪方式的热量差异巨大：

| 烹饪方式 | 每 100g 食材额外增加热量 | 评分影响 | 适用食材 |
|---------|----------------------|---------|---------|
| 生食/凉拌 | 0 kcal | 不变 | 蔬菜、部分海鲜 |
| 蒸 | 0 kcal | 不变 | 鱼、蛋、蔬菜、杂粮 |
| 煮/焯 | 0 kcal | 不变 | 蔬菜、面条、蛋 |
| 炖/焖 | 5-15 kcal | +0.5 | 肉类、根茎类 |
| 烤（不刷油） | 10-20 kcal | +0.5 | 鸡肉、鱼、红薯 |
| 炒（1 勺油） | 45-60 kcal | +1 | 大部分食材 |
| 煎（半煎炸） | 60-90 kcal | +1.5 | 蛋、鱼、肉饼 |
| 红烧 | 60-100 kcal | +1.5-2 | 肉类、鱼 |
| 炸 | 150-250 kcal | +3-4 | 所有食材 |

> 一勺油（10ml）= 约 90 kcal。很多家常炒菜实际用油 2-3 勺，仅油就增加了 180-270 kcal。

## 低油烹饪技巧

### 蒸

最适合减脂的烹饪方式，保留营养且不额外加油。

| 食材 | 时间 | 调味建议 |
|------|------|---------|
| 鱼（整条） | 8-10 分钟 | 葱姜丝 + 蒸鱼豉油（蒸好后再淋） |
| 鸡胸肉 | 12-15 分钟 | 葱姜 + 料酒，撕丝后拌酱油醋 |
| 蛋（水蒸蛋） | 8-10 分钟 | 蛋液:水 = 1:1.5，少量盐 |
| 蔬菜 | 3-5 分钟 | 蒜蓉 + 少量蚝油 |
| 红薯/玉米 | 15-20 分钟 | 直接蒸，无需调味 |

### 水炒法

用少量水代替油来炒菜，口感接近炒菜但热量大幅降低。

1. 锅热后加 2-3 勺水（不是油）
2. 放入蒜末/姜丝爆香
3. 加入蔬菜翻炒
4. 盖锅盖焖 1-2 分钟
5. 开盖收汁，调味即可

> 适合：叶菜类、西兰花、豆角等。不适合需要焦香口感的菜。

### 减油炒菜法

必须用油炒时，用以下方法减少用油量：

| 技巧 | 效果 |
|------|------|
| 不粘锅 | 可减少 50% 用油 |
| 油喷壶 | 一喷只加约 1-2g 油（普通倒油一勺约 10g） |
| 先焯后炒 | 蔬菜先焯水 8 成熟，再快速翻炒调味，只需少量油 |
| 用高汤/水代替部分油 | 炒菜中途加 1-2 勺水代替补油 |

## 调味替代方案

### 减盐不减味

| 替代方案 | 用法 |
|---------|------|
| 柠檬汁/醋 | 提供酸味，减少酱油用量 |
| 蒜/姜/葱 | 增香，减少对盐的依赖 |
| 黑胡椒/白胡椒 | 增加风味层次 |
| 香菜/芹菜/洋葱 | 天然增香 |
| 低钠酱油 | 钠含量减少 30-50% |
| 天然鲜味（蘑菇、海带、番茄） | 替代味精和鸡精 |

> 中国居民膳食指南建议每日盐摄入 < 5g。实际中国成年人平均每天 10-12g。减盐目标：先降到 7g/天，再逐步到 5g/天。

### 减糖不减味

| 场景 | 替代方案 |
|------|---------|
| 红烧菜需要炒糖色 | 减半或不加，用老抽上色 |
| 凉拌菜需要加糖 | 用少量苹果丝/梨丝提供自然甜味 |
| 煲汤加糖提鲜 | 不加，用番茄或玉米的自然甜味 |
| 腌制需要糖 | 减半，用少量蜂蜜替代白糖 |

### 常用酱料热量

| 酱料 | 每勺（15ml）热量 | 替代建议 |
|------|----------------|---------|
| 酱油 | ~10 kcal | 可用，控量 |
| 蚝油 | ~20 kcal | 减半使用 |
| 豆瓣酱 | ~15 kcal | 减半使用 |
| 芝麻酱 | ~90 kcal | 用酱油醋汁替代 |
| 沙拉酱（千岛酱） | ~80 kcal | 用油醋汁（醋 + 少量橄榄油） |
| 番茄酱 | ~20 kcal | 注意含糖量，用新鲜番茄替代 |
| 甜面酱 | ~25 kcal | 少量使用 |
| 老干妈/辣椒酱 | ~25 kcal | 注意含油量 |

## 一周健康烹饪示例

| 日期 | 早餐（≤ 10 分钟） | 午餐 | 晚餐（≤ 30 分钟） |
|------|------------------|------|------------------|
| 周一 | 燕麦粥 + 水煮蛋 + 苹果 | 外食/外卖 | 清蒸鱼 + 白灼菜心 + 杂粮饭 |
| 周二 | 全麦面包 + 无糖酸奶 + 蓝莓 | 外食/外卖 | 水炒西兰花 + 鸡胸肉丝 + 红薯 |
| 周三 | 蒸蛋羹 + 杂粮馒头 + 番茄 | 便当（前一晚备菜） | 番茄豆腐汤 + 凉拌黄瓜 + 玉米 |
| 周四 | 燕麦粥 + 坚果 + 猕猴桃 | 外食/外卖 | 清蒸鸡腿肉（去皮）+ 蒸南瓜 + 焯拌菠菜 |
| 周五 | 全麦面包 + 煮蛋 + 牛奶 | 外食/外卖 | 白灼虾 + 蒜蓉蒸茄子 + 杂粮饭 |
| 周六 | 杂粮粥 + 茶叶蛋 + 水果 | 放纵餐 | 清淡晚餐（汤 + 蔬菜） |
| 周日 | 燕麦 + 酸奶 + 水果 | 备菜日：准备周一至周三便当 | 蒸鱼/蒸鸡 + 蔬菜 + 杂粮饭 |

## 备菜技巧（Meal Prep）

适合工作日没时间做饭的人：

### 周末一次性准备

1. **杂粮饭** — 一次煮 3-4 天的量，分盒冷藏
2. **蛋白质** — 一次蒸/煮好鸡胸肉或白煮蛋，冷藏保存 3 天
3. **蔬菜预处理** — 洗好切好放保鲜盒，工作日快炒/焯水即可
4. **调味汁** — 提前调好酱油醋汁，装小瓶

### 保存时限

| 食物 | 冷藏保存 | 冷冻保存 |
|------|---------|---------|
| 煮好的米饭 | 3 天 | 1 个月 |
| 煮好的鸡胸肉 | 3 天 | 1 个月 |
| 水煮蛋（带壳） | 5 天 | 不建议 |
| 洗好的蔬菜 | 2-3 天 | 不建议 |
| 调味汁 | 1 周 | 不建议 |
