## 这是什么？

这是一组 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) 的 **Skill（技能插件）**，可以帮你：

- 根据体检数据做 **健康评估**
- 用 **食物评分系统** 灵活搭配饮食（不需要死记固定食谱）
- 生成个性化的 **4 周减脂计划**
- **跟踪进度**，每次回访后自动调整方案
- 用 **数据仪表盘** 直观查看体重和指标趋势

> 本工具基于公开医学指南生成，仅供参考，不构成医疗建议。请在医生指导下进行健康管理。

## 适合谁？

- 体检发现 **甘油三酯偏高**
- 有 **脂肪肝**（轻/中/重度）
- 年龄 **25-50 岁**，想科学减脂
- 想减脂但不知道怎么吃、怎么动

## 快速开始

### 前置条件

- 已安装 [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code)
- 有一个可用的 Claude 账号

### 安装

```bash
# 1. 克隆项目
git clone https://gitee.com/Ethanzyc/lose-fat.git
cd lose-fat

# 2. 在 Claude Code 中打开项目
claude
```

### 使用

在 Claude Code 对话中输入：

```
/lose-fat
```

然后跟着引导操作即可——回答一些基本的健康问题，系统会自动为你生成评估和减脂计划。

## 功能介绍

### `/lose-fat` — 健康管理主流程

| 功能 | 说明 |
|------|------|
| 健康评估 | 根据体检指标（TG、ALT、BMI 等）评估代谢风险 |
| 食物评分 | 63 种常见食物打分，自由搭配每餐，不再死记食谱 |
| 4 周计划 | 生成饮食 + 运动 + 生活习惯的综合方案 |
| 进度跟踪 | 定期回访，分析体重趋势，区分「水重」和「脂重」 |
| 多用户支持 | 可以为家人朋友分别创建档案 |

**输入方式二选一：**

- **引导问答**（推荐）— 逐步提问，每次 1-2 个问题
- **模板填写** — 复制模板一次填完粘贴回来

### `/lose-fat-dashboard` — 数据可视化

| 模式 | 说明 |
|------|------|
| 本地仪表盘 | 启动本地服务，浏览器中查看交互式趋势图表 |
| 静态报告 | 生成 HTML 文件，可离线查看或分享给他人 |

## 常见问题

**Q: 需要准备什么数据？**

A: 最少只需要：身高、体重、甘油三酯（TG）。其他指标（胆固醇、肝功能、血糖）有就提供，没有也不影响基本方案生成。

**Q: 减脂计划是固定食谱吗？**

A: 不是。采用 **食物评分系统**——每种食物有固定分数，只要每餐总分在目标范围内就可以自由搭配。比固定食谱更灵活，更容易坚持。

**Q: 数据保存在哪里？**

A: 所有数据保存在本地 `.lose-fat/profiles/` 目录，不会上传到任何服务器。

**Q: 可以多人使用吗？**

A: 可以。系统支持多用户档案，每个人独立管理。
