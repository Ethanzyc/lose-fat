## 这是什么？

这是一组 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) 的 **Skill（技能插件）**，让 Claude 变成你的专属减脂健康顾问。

- 根据体检数据做 **健康评估**（TG、BMI、脂肪肝、代谢综合征风险）
- 用 **食物评分系统** 灵活搭配饮食（63 种食物打分，自由组合，不需要死记固定食谱）
- 生成个性化的 **4 周减脂计划**（饮食 + 运动 + 生活习惯）
- **跟踪进度**，分析体重趋势，区分「水重」和「脂重」，正向鼓励
- 用 **数据仪表盘** 直观查看体重和指标趋势图表
- 支持日常随时聊天——记录体重、饮食、运动，提问营养和健康问题

> 本工具基于公开医学指南生成，仅供参考，不构成医疗建议。请在医生指导下进行健康管理。

## 适合谁？

- 体检发现 **甘油三酯偏高**（TG > 1.7 mmol/L）
- 有 **脂肪肝**（B 超诊断轻/中/重度）
- 合并 **代谢综合征** 风险（腹型肥胖、高血压、高血糖）
- 年龄 **25-50 岁**，想科学减脂
- 想减脂但不知道怎么吃、怎么动、怎么坚持

## 功能详解

### `/lose-fat` — 健康管理主流程

#### 1. 健康评估

输入体检指标后自动生成评估报告：

- **BMI 评估**：采用中国标准（偏瘦 < 18.5 / 正常 18.5-23.9 / 超重 24.0-27.9 / 肥胖 ≥ 28）
- **甘油三酯分级**：正常 < 1.7 / 边缘升高 1.7-2.3 / 升高 ≥ 2.3
- **脂肪肝分度**：结合 B 超结果和肝功能（ALT/AST）综合判断
- **代谢综合征筛查**：满足 3 项及以上指标异常时自动提示
- **历史趋势对比**：有多次体检数据时展示变化百分比

#### 2. 食物评分系统

不再使用固定食谱，而是给 63 种常见食物打分：

- **绿灯食物**（≤ 3 分）：推荐多吃（糙米、鸡胸肉、西兰花、苹果等）
- **黄灯食物**（3-5 分）：控制份量（红薯、去皮鸡腿、香蕉等）
- **红灯食物**（> 5 分）：严格限制（五花肉、薯片、含糖饮料等）
- 根据你的 TDEE 自动计算每餐目标分数，自由组合食物到目标范围即可

涵盖类别：谷物/主食（14 种）、蛋白质（14 种）、蔬菜（10 种）、水果（8 种）、奶制品（5 种）、油脂/坚果（6 种）、饮品/调味/零食（8 种）

#### 3. 4 周减脂计划

每次生成 4 周计划，包含三个维度：

- **饮食方案**：基于食物评分系统，给出每餐搭配示例和目标分数
- **运动方案**：根据 BMI 分层推荐（肥胖 → 散步/慢走、超重 → 快走/走跑交替、正常 → 慢跑/力量训练）
- **生活习惯**：2-3 个可执行的习惯调整目标

4 周周期结束后，基于最新数据自动生成下一个周期计划。

#### 4. 日常对话（随时可用）

不需要进入正式流程，随时聊天即可：

| 你说的话 | 系统会做什么 |
|---------|------------|
| `今天 75.2kg` | 自动记录体重，回复趋势变化 |
| `中午吃了碗牛肉面` | 自动记录饮食，给出评分和建议 |
| `晚上吃了火锅` | 自动记录，给出火锅类饮食建议 |
| `今天快走了 30 分钟` | 记录运动，给予鼓励 |
| `西瓜能吃吗？` | 基于知识库回答 |
| `平台期了怎么办` | 分析原因，给出突破策略 |
| `膝盖有点不舒服` | 调整运动建议 |

#### 5. 进度回访

定期回访时系统会：

- 分析体重趋势，区分「水重」（1-3 天快速波动）和「脂重」（持续变化）
- 检测平台期（连续 2 周以上体重不变），给出突破建议
- 统计饮食模式（外食频率、高碳水次数等）
- 无论结果如何都给予正向鼓励，关注非体重维度的进步
- 根据进展调整下一阶段方案

### `/lose-fat-dashboard` — 数据可视化

| 模式 | 说明 |
|------|------|
| 本地仪表盘 | 启动本地 Web 服务，浏览器中查看交互式趋势图表 |
| 静态报告 | 生成独立 HTML 文件，可离线查看或分享 |

仪表盘功能：

- BMI、甘油三酯、体重、体重变化四张概览卡片
- 体重趋势折线图（含目标体重参考线）
- 甘油三酯趋势折线图（含正常上限参考线）
- 时间范围筛选：全部 / 三年 / 一年 / 半年 / 三个月 / 一个月 / 一周 / 自定义日期区间
- 概览卡片跟随时间区间联动

## 常见问题

**Q: 需要准备什么数据？**

A: 最少只需要：身高、体重、甘油三酯（TG）。其他指标（胆固醇、肝功能、血糖）有就提供，没有也不影响基本方案生成。

**Q: 减脂计划是固定食谱吗？**

A: 不是。采用 **食物评分系统**——每种食物有固定分数，只要每餐总分在目标范围内就可以自由搭配。比固定食谱更灵活，更容易坚持。

**Q: 数据保存在哪里？**

A: 所有数据保存在本地 `.lose-fat/profiles/` 目录中的 JSON 文件，不会上传到任何服务器。

**Q: 可以多人使用吗？**

A: 可以。系统支持多用户档案，每个人独立管理，切换时选择对应姓名即可。

**Q: 我是 Mac/Windows/Linux，都能用吗？**

A: 都可以。只要能运行 Claude Code 就行。仪表盘使用纯 Python 标准库，无需额外安装依赖。
